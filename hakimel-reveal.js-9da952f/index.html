Streams

class UsingStreams{
  public static void main(String... args){
    List<String> strings = Arrays.asList("java", "perl", "clojure", "js");

    strings.stream()
           .filter( string -> string.startsWith("j") )
           .map( String::toUpperCase )
           .forEach( System.out::println );
  }
}
ao final desta manipulação, a lista original não se altera

Operações Intermediárias

  1. map → Transforma os dados
  2. flatMap → Transforma e normaliza os dados
  3. filter → Seleciona dados
  4. reduce → Agrega os dados

Operações Terminais

  1. forEach → itera nas posições da stream
  2. collect → monta uma estrutura baseada nos dados
  3. findFirst → recupera o primeiro valor
